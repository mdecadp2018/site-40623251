Week6

將倉儲中的 flaskapp.py 更新至最新版本

搜尋 github cmsimfly

git clone cmsimfly 的倉儲 git clone https://github.com/chiamingyen/cmsimfly

將 cmsimfly 裡的 flaskapp.py 複製貼上至自己的資料夾，更新 flaskapp.py

在 Leo 中，開啟 cmsimfly.leo

在 Leo 中的編輯 flaskapp.py 下按 ctrl + i 新增一個 @auto flaskapp.py

鼠標在 @auto flaskapp.py 上，按右鍵點 Refresh from disk

將原本的 @clean flaskapp.py 刪掉

把 @auto flaskapp.py 改為 @clean flaskapp.py
